# v-bind

v-bind 是單向資料綁定 (one-way data binding)的實作，是當綁定的變數值改變時，會重新將值帶入，進行渲染(render)

props / data

binding with object

# v-model

v-model 是雙向資料綁定 (two-way data binding)
